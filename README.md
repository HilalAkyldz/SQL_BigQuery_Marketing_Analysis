# 📊 Digital Marketing Data Analysis Project

## 📌 Project Overview
This project focuses on analyzing digital marketing data from multiple sources, including Facebook Ads, Google Ads, and GA4 (Google Analytics 4). The main goal is to evaluate campaign performance and understand user behavior through SQL-based analysis.

## 🎯 Objectives
- Analyze marketing campaign performance
- Identify high-performing campaigns
- Understand user behavior across the funnel
- Calculate marketing efficiency metrics such as ROMI
- Detect drop-off points in the conversion funnel

## 🗂️ Data Sources
- Facebook Ads data  
- Google Ads data  
- GA4 BigQuery dataset  

## 🛠️ Tools & Technologies
- SQL (DBeaver)
- Google BigQuery
- Data Analysis Techniques

## 🔍 Key Analysis Performed
- Daily spend analysis (average, minimum, maximum)
- ROMI (Return on Marketing Investment) calculation
- Best performing campaign analysis
- Monthly reach growth analysis
- Ad set performance consistency (streak analysis)
- Funnel analysis (session → purchase)
- Landing page conversion rate analysis

## ⚙️ Data Processing
- Data merging using `UNION`
- Table joining using `JOIN`
- Aggregations (`SUM`, `AVG`, `MIN`, `MAX`)
- Advanced analytics using Window Functions

## 📈 Key Insights
- Some campaigns significantly outperform others
- Conversion rates differ across traffic sources
- Funnel analysis shows user drop-offs at specific stages
- Certain landing pages have higher conversion rates

## 🧠 Learnings
- Performing data analysis using SQL
- Working with large datasets
- Understanding funnel and conversion analysis
- Making data-driven decisions

## 🚀 Future Improvements

- Perform deeper user segmentation
- Enhance data visualization for better insights
